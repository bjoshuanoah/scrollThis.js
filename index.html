    Scroll contents inside a div the easy way with scrollThis.js!

    Project maintained by bjoshuanoah Hosted on GitHub Pages — Theme by mattgraham

    Welcome to scrollThis.js

    Join the revolution! A clean scroller that doesn't break after 30 seconds on the site or web app. No frills, it just works. Easy installation:

    Installation

    1. Place the scrollThis.js file into your javascript folder, and reference the file after jQuery, jQueryui, and jQuery.mousewheel, but before any other event or function JS files.
    2. Make sure the div you want to scroll is inside container div. scrollThis only scrolls the first element inside of a div, so if you want to scroll a list, make sure to reference the container of the <ul>, not the <u\l> itself.
    3. Make sure the container div has a set (max)height, and has overflow set to hidden. We will not take care of this for you.
    4. Load your scrollThis() function inside of $(document).ready();. Reference the element using jQuery $('#container') as the argument. Either inside of another JS file, or directly in the markup as follows: 
       scrollThis($('#container'));

    If you have any questions, make sure to check out the example files.

    Dependancies

    1. jQuery
    2. jQueryui(you can use the custom minified version that includes JUST draggable events)
    3. jQuery.mousewheel

    All of these are included in the repo

    What about mobile?

    Mobile can take care of itself. How? in general mobile browsers take care of scrolling divs very well. We're not adding any functionality here; no drag to refresh or anything. This is a simple lightweight plugin specifically for getting rid of ugly scrollbars on desktop browsers.
